# API Testing with Postman

This section demonstrates API testing skills using Postman against a public REST API.

## Covered Scenarios
- Health checks
- CRUD operations on users
- Positive and negative test cases
- Status code and schema validation
- Authentication and authorization behavior

## Tools Used
- Postman
- JavaScript assertions
- Environment variables

## Notes
All requests were validated both via Postman and browser network inspection using Chrome DevTools.
